## Team Collaboration

- üí° Recommended: **[Microsoft Teams](https://teams.microsoft.com)**, **[Slack](https://slack.com)**
- Allows you to:
  - Create multiple channels for communication
  - Highly accessible as it's available in the browser.
  - Collaborate with external suppliers and contractors
  - You can integrate slack with Azure DevOps
    - E.g. by installing [Azure Pipeline app for Slack](https://docs.microsoft.com/en-us/azure/devops/pipelines/integrations/slack).
      1. Log-in through app
      2. Run `/azpipelines subscribe [project url]` inside a channel

## Modern DevOps Toolchain (2025)

### Identity & Authentication

- **Microsoft Entra ID** (formerly Azure Active Directory): Modern identity and access management
- **GitHub**: Git hosting and collaboration platform
- **OIDC (OpenID Connect)**: Secretless authentication for CI/CD pipelines

### Code & Collaboration

- **GitHub Codespaces**: Cloud-based development environments
- **Azure Deployment Environments**: On-demand test environments for deployments
- **Microsoft Teams**: Team collaboration and communication

### Security & Compliance

- **GitHub Advanced Security for Azure DevOps**: Integrated security scanning
- **Microsoft Defender for Cloud**: DevOps security threat detection
- **Azure Key Vault**: Secrets, keys, and certificate management

### Monitoring & Observability

- **Azure Monitor**: Application and infrastructure monitoring
- **Application Insights**: Application performance monitoring
- **Azure Log Analytics**: Centralized log management and analysis
- **Grafana**: Advanced visualization and dashboards
